# Ideas

### Hand Drawn Notes

Users could hand draw custom notes in a margin, with custom colors. This would probably need to be an optional setting, as it would make the text have a fixed layout, so that the text could be mapped properly. It also may not be able to work with smaller mobile devices like phones, or have a sepeperate layout dedicated specifically to them. 

One possibility, is to have custom canvases, that act as popups for verses. This would mean that you would have no issue with the format, as all notes are in their own canvas.

Drawn notes would be vector based, and overlaid over the background, with transparency enabled for given colors.

Colors for drawing can be named and added to a notebook. They would technically be a third notebook entry type, though `biblio_json` may need to be modified to support custom file formats like this.
